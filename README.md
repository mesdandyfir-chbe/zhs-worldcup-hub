# zhs-worldcup-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含一系列可直接在浏览器中运行的静态页面，内容覆盖信息展示、工具、参考等不同类别，不针对任何特定网站或域名。

## 项目简介

本仓库的主要目的是收集和维护一批独立的 HTML 文件，方便用户通过 GitHub Pages 或其他静态托管服务直接访问。所有页面均为纯前端实现，无后端依赖，适合作为学习、参考或日常使用的资料库。

## 目录说明

```
zhs-worldcup-hub/
├── index.html          # 仓库首页 / 导航入口
├── pages/              # 存放各独立页面的子目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── img/
└── README.md           # 本文件
```

- `index.html`：仓库的默认首页，通常包含页面列表或导航。
- `pages/`：所有独立 HTML 页面均放置在此目录下，便于管理和引用。
- `assets/`：存放页面共用的样式表、脚本和图片文件。

## 页面归档说明

- 每个页面均为独立、完整的 HTML 文件，可直接在浏览器中打开。
- 页面之间可能存在链接跳转，但无强制依赖关系。
- 页面内容可能随仓库更新而调整或补充，建议定期关注仓库变化。
- 部分页面可能包含外部资源（如字体、图标库），请确保网络连通。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 功能在线访问（需自行开启并配置）。
